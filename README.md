# idea_bot

Telegram Mini App, который подсказывает недорогие идеи для встречи с друзьями.

## Проблема

70% молодых людей в РФ отказывались от встреч с друзьями из-за нехватки денег
(province.ru, 2026). Барьер не в том, что сложно договориться о времени — это
уже решают другие инструменты (When2meet и подобные). Барьер в деньгах и
отсутствии готовых недорогих вариантов "куда пойти".

## Что делает

1. `/start` — бот присылает кнопку «Открыть 🎯», которая открывает Mini App
2. В Mini App выбираешь бюджет (бесплатно / до 500 ₽ / до 1500 ₽)
3. Получаешь одну конкретную идею (не список, чтобы не было паралича выбора)
4. Нативная кнопка Telegram внизу экрана — «Ещё вариант» (другая идея в том же бюджете)
5. Нативная кнопка «Назад» — вернуться к выбору бюджета

## Архитектура

- `bot.py` — Python, `python-telegram-bot` 21.6. Единственная задача — прислать
  кнопку, открывающую Mini App. Вся логика выбора идеи — в браузере.
- `docs/` — сам Mini App: `index.html` (разметка + стили + скрипт) и
  `ideas.js` (30 идей). Никакого бэкенда для Mini App не нужно — контент
  статичный.
- Подстраивается под тему Telegram (светлая/тёмная) через CSS-переменные
  `--tg-theme-*`, использует нативные `MainButton`/`BackButton` вместо
  HTML-кнопок — за счёт этого выглядит как часть Telegram, а не веб-страница.

## Деплой

**Mini App (docs/):** GitHub Pages — бесплатный HTTPS для статики
(Telegram принимает только HTTPS-ссылки для Mini App).
1. Запушить репозиторий на GitHub
2. Settings → Pages → Deploy from branch → выбрать ветку и папку `docs/`
3. Скопировать выданный URL (вида `https://<username>.github.io/<repo>/`)
4. Указать его в переменной окружения `WEBAPP_URL` при запуске бота

**Бот (bot.py):** Railway, как и mood-bot (см. `Procfile`).

## Запуск локально

```
pip install -r requirements.txt
export BOT_TOKEN=<токен от @BotFather>
export WEBAPP_URL=<URL после настройки GitHub Pages>
python bot.py
```

Проверить вёрстку Mini App без Telegram — просто открыть `docs/index.html`
в браузере (там свои резервные кнопки вместо нативных Telegram-кнопок).
