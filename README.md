# idea_bot

Telegram-бот, который подсказывает недорогие идеи для встречи с друзьями.

## Проблема

70% молодых людей в РФ отказывались от встреч с друзьями из-за нехватки денег
(province.ru, 2026). Барьер не в том, что сложно договориться о времени — это
уже решают другие инструменты (When2meet и подобные). Барьер в деньгах и
отсутствии готовых недорогих вариантов "куда пойти".

## Что делает бот

1. `/start` или `/idea` — бот спрашивает бюджет на компанию (бесплатно / до 500 ₽ / до 1500 ₽)
2. Выбираешь бюджет — получаешь одну конкретную идею (не список, чтобы не было паралича выбора)
3. «Ещё вариант» — другая идея в том же бюджете
4. «Другой бюджет» — вернуться к выбору бюджета

## Стек

- Python, `python-telegram-bot` 21.6
- Идеи — статичный список в `ideas.py`, без БД (контент неизменяемый)
- Деплой — Railway (см. `Procfile`)

## Запуск локально

```
pip install -r requirements.txt
export BOT_TOKEN=<токен от @BotFather>
python bot.py
```
